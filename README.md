This repository contains the code used in my final year project on predictive analysis of customer churn in telecommunications.

- `telco_churn_model.pdf`: exported notebook in PDF format

## Main techniques
- Data preprocessing
- Feature engineering
- Logistic Regression
- SVM

- XGBoost
- SHAP interpretability
